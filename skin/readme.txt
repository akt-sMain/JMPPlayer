
●スキン作成方法

1.カスタム画像を保管するSkinフォルダを作成してください。
2.defaultフォルダに入っている画像と同じファイル名でアイコンを作成してください。
3.フォルダにアイコンを追加。
4.「skin.txt」の"USE=default"のdefaultを作成したフォルダ名に変更してください。
5.ソフトを再起動してください。

=##=

● How to create a skin

1. Create a Skin folder to store your custom images.
2. Create an icon with the same file name as the image in the default folder.
3. Add an icon to the folder.
4. Change the default of "USE=default" of "skin.txt" to the created folder name.
5. Restart the software.
